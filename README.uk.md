# sub-deck

Плагін для Decky Loader на Steam Deck, що дозволяє зручно керувати підписками VLESS, VMess, Trojan, Shadowsocks та Hysteria 2, а також підключатися до них з автоматичним системним TUN-тунелюванням усього трафіку через `sing-box`.

![Screenshot](assets/screenshot.png?v=2)

## 📋 Можливості

- **Підтримка безлічі протоколів**: VLESS (Reality, TLS, None), VMess, Trojan, Shadowsocks та Hysteria 2 / Hy2.
- **Кілька підписок**: Можливість додавати та керувати різними посиланнями на підписки одночасно. Сервери з усіх підписок об'єднуються в загальний список.
- **Підключення в один клік**: Натисніть на сервер у списку для підключення, натисніть повторно на активний сервер — для відключення. Більше жодних зайвих тумблерів.
- **Системний TUN-режим**: Автоматично перенаправляє весь інтернет-трафік Steam Deck через інтерфейс `sing-box` TUN (`tun0`).
- **Режими маршрутизации**: Вибір між режимом **«За замовчуванням»** (весь трафік йде через VPN) та **[RoscomVPN](https://github.com/hydraponique/roscomvpn-routing)** (розумний обхід блокувань: Steam, Twitch та ресурси UA/RU/BY напряму, заблоковані сайти через VPN, реклама та телеметрія Windows блокуються).
- **Швидкий експорт логів**: Кнопка **LOG** об'єднує логи плагіна та ядра, записує їх у `/home/deck/sub-deck.log` та копіює в буфер обміну без спливаючих вікон.
- **Усе включено**: Автоматично завантажує стабільний бінарний файл `sing-box` при першому підключенні.

## 🎁 Вбудовані безкоштовні підписки

У плагін додано кнопку **«Безкоштовні підписки»**, яка в один клік додає три перевірені та автоматично оновлювані джерела ключів:
- [Підписка igareck](https://github.com/igareck/vpn-configs-for-russia): Класичне перевірене джерело конфігурацій для обходу блокувань.
- [Goida VPN (AvenCores)](https://github.com/AvenCores/goida-vpn-configs): Часто оновлювані конфігурації VLESS від проєкту Goida VPN.
- [Підписка zieng2/wl](https://github.com/zieng2/wl): Якісні універсальні конфігурації VLESS із репозиторію wl від zieng2.

*Примітка: Для забезпечення максимальної швидкості та продуктивності, для безкоштовних підписок автоматично вибираються та відображаються лише топ-5 найшвидших робочих нод за результатами паралельного TCP-пінгу.*

## 📥 Встановлення

1. Завантажте останню версію (`sub-deck.zip`) зі сторінки релізів або зберіть її вручную.
2. Скопіюйте ZIP-архів на ваш Steam Deck.
3. Увімкніть **Режим розробника** в системних налаштуваннях Steam Deck. Потім у налаштуваннях Decky Loader перейдіть до розділу розробника, активуйте **Developer mode** та виберіть пункт «Встановити плагін з файлу».

## 🚀 Використання

1. Вставте ваше посилання на підписку в текстове поле та натисніть **«Додати підписку»**.
2. Натисніть на будь-який сервер у списку **«Доступні сервери»**, щоб активувати VPN.
3. Підключений сервер підсвітиться синім кольором, а зверху з'явіться напис **«Підключено до: <ім'я>»**.
4. Щоб відключитися, просто натисніть на цей же сервер у списку ще раз.
