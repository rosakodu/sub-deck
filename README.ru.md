# sub-deck

[English](README.md)

Плагин для Decky Loader на Steam Deck, позволяющий удобно управлять подписками VLESS, VMess, Trojan, Shadowsocks и Hysteria 2, а также подключаться к ним с автоматическим системным TUN-туннелированием всего трафика через `sing-box`.

![Screenshot](assets/screenshot.png)

## 📋 Возможности

### 🛠️ Режимы маршрутизации (Пресеты)
*   **«По умолчанию» (Default)** — весь трафик идет через VPN.
*   **«RoscomVPN»** — умный обход блокировок (сайты РФ/РБ и BitTorrent напрямую, заблокированные ресурсы через VPN, реклама и телеметрия Windows блокируются).
*   **Офлайн-базы и автообновление:** Автоматическое скачивание и обновление гео-баз силами Python в фоне раз в 6 часов (VPN стартует мгновенно благодаря встроенным базам).

### 🔍 Интеллектуальная фильтрация серверов
*   **Быстрый пинг:** Параллельный TCP-пинг при загрузке подписки (таймаут 1.0 сек).
*   **Топ-15 лучших:** Автоматический выбор и отображение топ-15 самых быстрых рабочих нод (даже если в подписке более 100 000 ключей, для бесплатных конфигураций лимит составляет 5 нод).
*   **Дедупликация:** Автоматическое удаление повторяющихся серверов с одинаковыми именами или адресами.

### 📱 Обновленный вертикальный интерфейс
*   **Группировка:** Серверы больше не смешиваются в общую кучу, а сгруппированы строго внутри своих подписок.
*   **Удобное управление:** Кнопка «Обновить» расположена вертикально над кнопкой «Удалить» для каждой подписки.
*   **Компактный дизайн:** Добавлен компактный шрифт (11px, uppercase) для заголовков подписок.
*   **Идеальное выделение:** Обводка выбранной ноды и режима маршрутизации ложится пиксель-в-пиксель по контуру кнопок.

### 🎁 Кнопка «Бесплатные конфигурации» (Free configs)
*   Возможность в один клик добавить проверенную и ежедневно обновляемую базу бесплатных ключей для РФ (VPN VLESS Configs Russia).

### 🔄 Фоновое автообновление подписок
*   Добавлена поддержка HTTP-заголовков `profile-update-interval` для автоматического обновления серверов в фоне.

## 📥 Установка

1. Скачайте последнюю версию (`sub-deck.zip`) со страницы релизов или соберите её вручную.
2. Скопируйте ZIP-архив на ваш Steam Deck.
3. Включите **Режим разработчика** в системных настройках Steam Deck. Затем в настройках Decky Loader перейдите в раздел разработчика, активируйте **Developer mode** и выберите пункт «Установить плагин из файла».

## 🚀 Использование

1. Вставьте вашу ссылку на подписку в текстовое поле и нажмите **«Добавить подписку»**.
2. Нажмите на любой сервер в списке **«Доступные серверы»**, чтобы активировать VPN.
3. Подключенный сервер подсветится синим цветом, а сверху появится надпись **«Подключено к: <имя>»**.
4. Чтобы отключиться, просто нажмите на этот же сервер в списке ещё раз.
