# sub-deck

Плагин для Decky Loader на Steam Deck, позволяющий удобно управлять подписками VLESS, VMess, Trojan, Shadowsocks и Hysteria 2, а также подключаться к ним с автоматическим системным TUN-туннелированием всего трафика через `sing-box`.

![Screenshot](assets/screenshot.png)

## 📋 Возможности

- **Поддержка множества протоколов**: VLESS (Reality, TLS, None), VMess, Trojan, Shadowsocks и Hysteria 2 / Hy2.
- **Несколько подписок**: Возможность добавлять и управлять различными ссылками на подписки одновременно. Серверы из всех подписок объединяются в общий список.
- **Подключение в один клик**: Нажмите на сервер в списке для подключения, нажмите повторно на активный сервер — для отключения. Больше никаких лишних тумблеров.
- **Системный TUN-режим**: Автоматически перенаправляет весь интернет-трафик Steam Deck через интерфейс `sing-box` TUN (`tun0`).
- **Быстрый экспорт логов**: Кнопка **LOG** объединяет логи плагина и ядра, записывает их в `/home/deck/sub-deck.log` и копирует в буфер обмена без всплывающих окон.
- **Всё включено**: Автоматически скачивает стабильный бинарный файл `sing-box` при первом подключении.

## 🎁 Встроенные бесплатные подписки

В плагин добавлена кнопка **«Бесплатные подписки»**, которая в один клик добавляет три проверенных и автоматически обновляемых источника ключей:
- [Подписка igareck](https://github.com/igareck/vpn-configs-for-russia): Классический проверенный источник конфигураций для обхода блокировок в РФ (ранее «Бесплатные конфигурации»).
- [Goida VPN (AvenCores)](https://github.com/AvenCores/goida-vpn-configs): Часто обновляемые конфигурации VLESS от проекта Goida VPN.
- [Подписка zieng2/wl](https://github.com/zieng2/wl): Качественные универсальные конфигурации VLESS из репозитория wl от zieng2.

*Примечание: Для обеспечения максимальной скорости и производительности, для бесплатных подписок автоматически выбираются и отображаются только топ-5 самых быстрых рабочих нод по результатам параллельного TCP-пинга.*

## 📥 Установка

1. Скачайте последнюю версию (`sub-deck.zip`) со страницы релизов или соберите её вручную.
2. Скопируйте ZIP-архив на ваш Steam Deck.
3. Включите **Режим разработчика** в системных настройках Steam Deck. Затем в настройках Decky Loader перейдите в раздел разработчика, активируйте **Developer mode** и выберите пункт «Установить плагин из файла».

## 🚀 Использование

1. Вставьте вашу ссылку на подписку в текстовое поле и нажмите **«Добавить подписку»**.
2. Нажмите на любой сервер в списке **«Доступные серверы»**, чтобы активировать VPN.
3. Подключенный сервер подсветится синим цветом, а сверху появится надпись **«Подключено к: <имя>»**.
4. Чтобы отключиться, просто нажмите на этот же сервер в списке ещё раз.
